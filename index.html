  • Nombre completo

Los datos personales recabados serán utilizados para las siguientes finalidades primarias:

  • Atender solicitudes de información y contacto

De manera adicional, sus datos podrán ser utilizados para las siguientes finalidades secundarias:

  • Envío de información comercial, promociones o publicidad

Si no desea que sus datos personales sean tratados para estas finalidades secundarias, puede manifestarlo enviando un correo a contacto.maiaeve@gmail.com

3. Transferencia de datos personales

MAIA no comparte ni transfiere sus datos personales a terceros, salvo cuando sea necesario para cumplir con obligaciones legales o requerimientos de autoridad competente.

4. Derechos ARCO

Usted tiene derecho a Acceder, Rectificar, Cancelar u Oponerse al tratamiento de sus datos personales (Derechos ARCO).

Para ejercerlos, deberá enviar una solicitud al correo electrónico contacto.maiaeve@gmail.com, incluyendo:

  • Nombre del titular
  • Derecho que desea ejercer
  • Descripción clara de los datos personales

Este sitio web puede utilizar cookies y otras tecnologías para mejorar la experiencia del usuario, analizar el tráfico y optimizar el contenido.

6. Cambios al aviso de privacidad
